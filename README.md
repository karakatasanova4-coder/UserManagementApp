# Система управления пользователями

## Описание
Данный проект представляет собой простую систему управления пользователями
для небольшой компании. Проект реализован на языке Java и демонстрирует
принципы YAGNI, KISS и DRY.

## Функциональность
- Добавление пользователей
- Удаление пользователей
- Обновление информации о пользователях

## Используемые технологии
- Java
- IntelliJ IDEA

## Принципы проектирования
- YAGNI (реализована только необходимая функциональность)
- KISS (код простой и понятный)
- DRY (исключено дублирование кода)

## Как запустить проект
1. Открыть проект в IntelliJ IDEA
2. Запустить файл Main.java
